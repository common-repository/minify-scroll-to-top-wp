* Go to your Dashboard after installation and navigate to "Appearance > Customize > Scroll To Top" to configure the accordion.

= Features =

  * Very easy installation
  * Flexible and easy to use


== Installation ==

1. Upload `minify-scroll-to-top-wp` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Customize Then Minify Scroll To Top Color change and Published
4. font awesome icon class( fa fa-angle-up ) Copy This and paste font awesome icon
